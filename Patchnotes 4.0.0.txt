This is a massive update that completely rewrites the bot's core architecture. We moved from a monolithic synchronous script to a modular, high-performance asynchronous application. It also introduces long-awaited features like Plex Login and a smart solution for Group Chat authentication.

## üö® BREAKING CHANGES
*   **Renamed Main Script:** The script `telegram_overseerr_bot_beta.py` has been renamed to **`main.py`**.
    *   **Action Required:** You must update your startup commands, systemd services, or custom scripts to run `python main.py`.
    *   **Docker Users:** If you pull the new image, this is handled automatically. If you use a custom Dockerfile, please update the `CMD`.
*   **Configuration Structure:** The project now uses a modular structure (`main.py`, `utils.py`, `overseerr_api.py`, `bot_settings.py`). Ensure you deploy **all** files, not just the main script.
